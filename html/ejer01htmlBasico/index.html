Cuerpo Principal del Contenido

La etiqueta main es en HTML5 una etiqueta Semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella .Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento
El contenido escrito aqui dentro no se repite en el resto del sitio(todo lo contrario a los encabezados de pagina, los menues de navegacion y los de pie de pagina). Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontrar la pagina luego de procesos de indexacion de los buscadores.

Seccion Etiquetas Semanticas Estructurales mas Importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las Etiquetas Semanticas Estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento

Reglas para Etiquetas Semanticas

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION.
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. Osea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden ser anidadas dentro de otros HEADER o FOOTER. Osea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento.
  • Reservar etiquetas h1 para usar dentro de etiquetas top level como MAIN, HEADER o FOOTER

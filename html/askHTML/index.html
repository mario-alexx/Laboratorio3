  3. ¿Como se relaciona un header http con la etiqueta head de HTML?

    4. No hay una relación, son cosas distintas. Pero en cierto caso particular existe cierta relación, dentro del documento html se puede simular headers http con el atributo http-equiv.

  4. ¿Que es UTF8?

    5. Es un formato estándar para almacenar caracteres Unicode, utiliza una secuencia única de 1, 2, 3 o 4 bytes (longitud variable) para codificar cada carácter en el juego de caracteres Unicode.

  6. ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestana?

  8. ¿Que atributos de estilo basicos aplicarian a una tabla? Explicar.

  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    10. Todo. El body html representa el contenido de un documento html y el body http es un campo que lleva los datos asociados con la petición, los archivos o documentos asociados a una respuesta.

  10. ¿Que atributos marcan la agrupacion de celdas de una tabla?. ¿A que etiquetas se aplican?
